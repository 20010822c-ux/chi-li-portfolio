# Chi Li Portfolio

李持 Chi Li 的中文个人作品集网站，使用 Next.js、TypeScript 与 Tailwind CSS 构建，部署目标为 Vercel。

## 开发

```bash
npm install
npm run dev
```

打开 <http://localhost:3000> 查看网站。

## 构建

```bash
npm run build
```
